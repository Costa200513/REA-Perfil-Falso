# Gram – Perfil ficticio para análisis de privacidad digital

**Gram** es un recurso educativo abierto desarrollado con HTML, CSS y JavaScript que simula un perfil ficticio de una red social. Su objetivo es servir como material didáctico para trabajar temas vinculados con la privacidad digital, la imagen en línea, la huella digital y la información que puede inferirse a partir de publicaciones, historias y datos visibles en redes sociales.

El perfil presentado es completamente ficticio. No pertenece a una persona real. Todos los nombres, publicaciones, historias destacadas, imágenes, textos y datos fueron creados o seleccionados con fines educativos.

---

## Propósito del recurso

El propósito principal de este recurso es ayudar a estudiantes, docentes y otros usuarios a reflexionar sobre cómo un perfil de red social puede revelar información personal, incluso cuando esa información no aparece escrita de forma directa.

A partir de la exploración del perfil ficticio, se pueden analizar aspectos como:

- información visible en publicaciones e historias;
- datos personales que pueden inferirse;
- rutinas, horarios y lugares frecuentes;
- vínculos sociales;
- exposición de emociones, actividades y espacios personales;
- riesgos asociados a la privacidad, la seguridad y la imagen digital;
- recomendaciones para publicar de forma más segura.

---

## Uso educativo

Este recurso puede utilizarse en clases, talleres, cursos virtuales, actividades de reflexión o propuestas interdisciplinarias relacionadas con ciudadanía digital.

Está pensado especialmente para trabajar contenidos vinculados con:

- privacidad digital;
- seguridad en redes sociales;
- huella digital;
- identidad e imagen en línea;
- análisis crítico de publicaciones;
- uso responsable de redes sociales;
- protección de datos personales.

El recurso puede ser usado directamente o adaptado según las necesidades de cada docente, grupo, nivel educativo o contexto institucional.

---

## Actividades sugeridas

Algunas posibles actividades educativas con este recurso son:

- analizar qué información aparece directamente en el perfil;
- identificar qué información puede inferirse al relacionar publicaciones, imágenes e historias;
- reconocer lugares frecuentes, rutinas u horarios;
- discutir posibles riesgos asociados a la exposición de datos personales;
- proponer cambios para reducir la exposición del perfil;
- comparar publicaciones seguras e inseguras;
- elaborar recomendaciones para un uso responsable de redes sociales;
- adaptar el perfil ficticio para crear nuevos casos de análisis.

---

## Funcionalidades implementadas

La página incluye:

- perfil ficticio de una estudiante;
- nombre de usuario y biografía;
- contador de publicaciones, seguidores y seguidos;
- botón funcional de **Seguir / Siguiendo**;
- aumento y disminución del contador de seguidores;
- historias destacadas clickeables;
- visor de historias destacadas;
- publicaciones en grilla vertical;
- visualización ampliada de publicaciones al hacer clic;
- pistas educativas sobre privacidad digital;
- modo de análisis para observar información expuesta;
- modo claro y modo oscuro;
- diseño responsive adaptable a computadora, tablet y celular;
- sección de créditos del recurso educativo.

## Estructura del proyecto

```text
/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── img/
│   └── imágenes del recurso en formato .webp
└── README.md
```
Las imágenes deben estar ubicadas dentro de la carpeta `img` para que el sitio las cargue correctamente.

Se recomienda utilizar imágenes en formato `.webp`, ya que permiten reducir el peso de los archivos y mejorar la carga del sitio, especialmente en dispositivos móviles.

Si se modifican los nombres de las imágenes, también deben actualizarse las rutas correspondientes en el archivo `index.html` o en el archivo `script.js`.

## Cómo usar el recurso

1. Descargar o clonar este repositorio.
2. Abrir el archivo `index.html` en un navegador.
3. Explorar el perfil ficticio, las publicaciones y las historias destacadas.
4. Utilizar el recurso como disparador para actividades de análisis, debate o producción.

También puede publicarse fácilmente mediante GitHub Pages para compartirlo con estudiantes o docentes.

---

## Cómo adaptar el recurso

Este recurso está pensado para ser reutilizado y modificado libremente en contextos educativos.

Se puede adaptar modificando:

- los textos del perfil;
- la biografía;
- las publicaciones;
- las historias destacadas;
- las imágenes;
- las pistas de análisis;
- los colores y estilos visuales;
- las preguntas o consignas de trabajo;
- el nivel de dificultad de la actividad;
- el contexto del caso ficticio.

Por ejemplo, se puede transformar el perfil en un nuevo caso de análisis sobre otro personaje ficticio, otro nivel educativo, otra red social simulada o una situación diferente relacionada con privacidad digital.

---

## Publicación en GitHub Pages

El recurso puede publicarse como sitio web estático mediante GitHub Pages.

Para hacerlo:

1. Subir el proyecto completo a un repositorio de GitHub.
2. Verificar que `index.html` esté en la raíz del repositorio.
3. Ir a **Settings**.
4. Entrar en **Pages**.
5. Seleccionar la rama principal del proyecto.
6. Guardar los cambios.
7. Acceder al enlace generado por GitHub Pages.

---

## Aclaración importante

Este sitio no busca replicar oficialmente Instagram ni utilizar su identidad de marca. Se trata de una simulación educativa de una red social ficticia llamada **Gram**, creada únicamente para analizar prácticas de publicación, privacidad, seguridad digital e información inferida en entornos digitales.

Ninguna información presentada corresponde a una persona real.
